# 用户注册功能

## 启用方法

### 1. 配置支持多用户的存储

本地开发示例：

配置 `.env.local`：

```bash
NEXT_PUBLIC_STORAGE_TYPE=redis
REDIS_URL=redis://localhost:6379
```

生产部署示例：

在部署平台添加环境变量：

```bash
NEXT_PUBLIC_STORAGE_TYPE=upstash
UPSTASH_URL=your_upstash_url
UPSTASH_TOKEN=your_token
```

然后重新部署。

### 2. 在后台开启注册

1. 以站长账号登录 `/admin`
2. 展开“用户配置”
3. 在“公开注册”中按需选择“新注册用户默认用户组”
4. 开启“允许访客自行注册账号”

开关保存后立即生效，无需修改环境变量或重新部署。

## 功能说明

- 图形验证码防护
- 实时表单验证
- 用户名：3-20 字符
- 密码：至少 6 位

## 使用建议

- 公网开放注册前，先建立受限用户组并作为默认组
- 注册完成后在同一后台开关处及时关闭
- 定期查看用户列表并封禁异常账号

> 旧部署中的 `NEXT_PUBLIC_ENABLE_REGISTRATION` 和 `DEFAULT_REGISTRATION_GROUP` 仅用于首次迁移后台初始设置，后续请在管理面板操作。
